# Streamer / Affiliate Landing Page

A simple static landing page built with HTML, CSS, and JavaScript.

## How to use in VS Code

1. Open this folder in VS Code.
2. Edit these files:
   - `index.html` for words, links, and sections
   - `styles.css` for colors and design
   - `script.js` for small interactions
3. Right-click `index.html` and choose **Open with Live Server**.

## How to deploy to GitHub + Netlify

1. Create a new GitHub repository.
2. Upload/push these files.
3. Go to Netlify.
4. Click **Add new site** → **Import an existing project**.
5. Connect GitHub and select your repository.
6. Build command: leave blank.
7. Publish directory: `/` or leave blank.
8. Deploy.

## Things to change

Search for these placeholders and replace them:

- `YOURNAME`
- `INDAY`
- `https://example.com`
- `https://kick.com/yourname`
- Social media links
- Disclaimer text
